# Fruit Ripeness Detection Project

This project aims to detect fruit ripeness using gas sensors
and multi-channel ADC (MCP3008).

## Hardware
- Arduino Mega
- Raspberry Pi
- MCP3008 ADC
- MQ / TGS gas sensors

## Current Progress
- MCP3008 SPI communication verified
- Multi-channel ADC reading tested
- Data output in CSV format

## Folder Structure
- arduino/: Arduino source code
- raspberry_pi/: Raspberry Pi Python scripts
- docs/: project documents
